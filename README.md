# AppBlocker

Interfaz conceptual de AppBlocker diseñada con un acabado inspirado en iOS 17/18. El proyecto utiliza Flutter con componentes `Cupertino` para recrear la estética de Apple: tipografía SF, tarjetas translúcidas, gradientes suaves y espaciados generosos.

## Características clave

- **Navegación** con `CupertinoTabScaffold` y barras inferiores con iconografía SF Symbols.
- **Pantalla de resumen** con tarjetas de foco dinámicas, controles segmentados y seguimiento de apps bloqueadas.
- **Gestión de horarios** estilo iOS con tarjetas agrupadas, interruptores y botones rellenos.
- **Panel de insights** con métricas, gradientes y un control segmentado para alternar periodos.
- **Preferencias** con cabecera de perfil, toggles y enlaces secundarios siguiendo patrones de iOS.

## Requisitos

- Flutter 3.16 o superior.
- SDK de Dart 3.0 o superior.

## Puesta en marcha

```bash
flutter pub get
flutter run
```

> Nota: Este proyecto es una maqueta UI y no incluye lógica de negocio ni integraciones reales.
